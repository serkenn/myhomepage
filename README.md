# Kentaro Yamamoto - Profile Page

サイバーパンク/ネオン風デザインのポートフォリオ・プロフィールページです。

## URL

https://serken.tech/

## ローカルで確認

`index.html` をブラウザで直接開いてください。ビルドやサーバーは不要です。

## 機能

- Vanta.js によるアニメーション背景（ネットワークエフェクト）
- Vanilla Tilt による3Dカードエフェクト
- グラスモーフィズムデザイン
- 日本語 / 英語の言語切替
- レスポンシブ対応（スマホ / PC）

## 技術スタック

- HTML / CSS / JavaScript（単一ファイル構成）
- [Three.js](https://threejs.org/) + [Vanta.js](https://www.vantajs.com/)
- [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/)
- [Google Fonts (M PLUS Rounded 1c)](https://fonts.google.com/specimen/M+PLUS+Rounded+1c)
